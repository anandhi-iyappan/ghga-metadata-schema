    Class: sequencing protocol¶

    Information about the sequencing of a sample.

    URI: GHGA:SequencingProtocol

    Parents¶

    • is_a: Protocol - A plan specification which has sufficient level of detail and quantitative information to communicate it between investigation agents, so that different investigation agents will reliably be able to independently reproduce the process.

Can be one of several types; jet-in-air, quartz cuvette, or a hybrid of the two. The sample flows through the center of a fluid column of sheath fluid in the flow cell.
      • Range: String
